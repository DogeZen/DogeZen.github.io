1.python json存储时乱码

json.dumps默认用ascii码存储，,ensure_ascii=False后才能正常转。

另外就是文件打开时需要编码为utf-8

2.python url 解析特殊字符报错

url带空格，+, = , ?等特殊字符导致urllib.request.urlopen(url)解析失败。

这里不是python包的问题，是本身传过来的url就不应该带这类字符。

开发中出现这个问题，需要先去跟同事沟通，确定问题来源。
